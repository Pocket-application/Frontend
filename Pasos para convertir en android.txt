1. vite.config.js - base: './'
2. npm install @capacitor/android@latest @capacitor/core@latest @capacitor/cli@latest
3. npx cap init
4. npm install @capacitor/android
5. npx cap add android
6. npm run build
7. npx cap sync android
9. Crear una carpeta en la raíz llamada "assets" y copiar el favicon en assets nombralo icon-only
10. npm install @capacitor/assets
11. npx capacitor-assets generate --android
12. crear local.properties en la carpeta android con la siguiente línea: sdk.dir=C\:\\Android
13. android/app/build.grale
buildTypes {
        release {
            minifyEnabled false
            proguardFiles(
                getDefaultProguardFile('proguard-android-optimize.txt'),
                'proguard-rules.pro'
        )}
    }
14. en la carpeta android ejecutar .\gradlew clean
15. npm run build
16. npx cap sync android
17. añadir una línea en android\app\src\main\AndroidManifest.xml
<application
        android:allowBackup="true"
        android:icon="@mipmap/ic_launcher"
        android:label="@string/app_name"
        android:roundIcon="@mipmap/ic_launcher_round"
        android:supportsRtl="true"
        android:theme="@style/AppTheme"
        android:networkSecurityConfig="@xml/network_security_config">
18. Crear el archivo android\app\src\main\res\xml\network_security_config.xml
<?xml version="1.0" encoding="utf-8"?>
<network-security-config>
  <domain-config cleartextTrafficPermitted="false">
    <domain includeSubdomains="true">oscarpalomino.dev</domain>
  </domain-config>
</network-security-config>

17. npx cap run android
